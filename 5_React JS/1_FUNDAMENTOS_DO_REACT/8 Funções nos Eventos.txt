Quando as funções sao simples, podemos realizar a logica no próprio
evento;

Isso torna nossa código mais ‘complicado’, por atrelar logica com
HTML

Mas em algumas situações 6 aplicável;

Vamos ver na pratica!

Passo 1: vamos usar mesmo componente da aula 7, mas vamos usar uma função dentro do HTML veja:

const Events = () =>{
    
    const imprimirNoConsole = () => {
        console.log("O evento de click funcionou");
    }
    
    return(
        <div>
            <button onClick={imprimirNoConsole}>Clique aqui</button>
            <button onClick={(e) => console.log("Evento dentro do HTML")}>Evento no HTML</button> //ficaria mais confuso se a função precisasse ser maior
        </div>
    );
};

export default Events;
